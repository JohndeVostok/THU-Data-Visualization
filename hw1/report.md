# Homework1 Report

## 问题简述

### 疫情数据动态气泡图

可视化每一个国家的疫情发展情况，图元颜色代表国家，横轴、纵轴和图元半径（面积）所代表意义自选。

在我的实现中，横坐标表示累计确诊人数，纵坐标表示死亡人数，半径表示当前确诊人数

### 疫情数据动态折线图

通过一张动态折线图反应不同国家累计确诊人数的变化趋势，要求随着最大数量的变化，纵坐标范围可以相应自动缩放

在我的实现中，横坐标固定，随着时间推进，纵坐标动态变化

### 简答

在动态气泡图的实现中，首先是横纵坐标的选择，我认为累计确诊和死亡人数，是能够表现疫情烈度和惨度最直观的两个数值，两项均为累计数值。因此直接选择作为横纵坐标我认为是合适的。而半径我选择了当前确诊人数，这一项表示当前时间疫情的发展程度他的放大和缩小能够比较好的体现疫情应对的状况。

## 实现

### 设计

我使用matplotlib作为可视化的工具。简单来说就是每一跳分别绘制一张图，每张图之间间隔0.1s，实现上非常简单。

实现过程中除matplotlib文档外，没有参考任何资料。

### 使用方法

存入world.csv

``` shell
    cp world.csv hw1/.
```

生成序列化数据

``` shell
    python data.py
```

生成动态气泡图

``` shell

```

``` shell
    python line.py
```
